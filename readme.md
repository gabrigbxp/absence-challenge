# How to run project

- Install using [yarn](https://classic.yarnpkg.com/en/docs/install) documentation
- Run using `yarn dev`
- Environment variables required
  1. JWT_SECRET: secret for JSON Web Token
  2. PORT: port for server (8000 by default)
